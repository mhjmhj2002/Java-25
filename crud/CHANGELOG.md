# Changelog

Todas as mudanças relevantes deste projeto serão documentadas neste arquivo.

O versionamento segue Semantic Versioning (SemVer).

---

## [0.5.0] - 2026-06-27

### Added

- Testes da camada REST utilizando MockMvc
- Cobertura dos endpoints principais
- Testes de criação de produto
- Testes de busca por produto inexistente
- Testes de validação de requisições
- Testes de atualização
- Testes de remoção

### Improved

- Organização da suíte de testes
- Separação entre testes unitários e testes REST

---
